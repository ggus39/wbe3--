# 🌐 LearnVerse

> **Web3 知识共享与激励平台**：构建由社区所有的去中心化知识库。

## 🏗️ 项目架构
- 📂 **contracts/**: 智能合约 (Solidity)
- 📂 **frontend/**: 交互界面 (Next.js)
- 📂 **content/**: 知识库核心 (Markdown)
- 📂 **docs/**: 协议与指南

## 🛠️ 技术栈
- **前端**: Next.js 14, Tailwind CSS
- **合约**: Solidity, Foundry
- **存储**: IPFS / Arweave
