# wbe3--
A decentralized Web3 knowledge sharing platform.
# 🌐 LearnVerse: Open-Source Web3 Knowledge Hub

> **使命**：通过去中心化的方式，构建一个由社区驱动、所有权归属于贡献者的 Web3 知识库。

---

## 🏗️ 项目架构 (Directory Structure)

我们采用标准的多包管理结构，方便不同背景的贡献者参与：

* 📂 **`contracts/`**: 存放智能合约。我们计划使用灵魂绑定代币 (SBT) 来证明用户的学习成就。
* 📂 **`frontend/`**: 核心 Web 应用程序，基于 Next.js 构建，支持钱包登录。
* 📂 **`content/`**: 知识库核心！所有 Markdown 格式的 Web3 教程、案例分析都放在这里。
* 📂 **`docs/`**: 项目的设计文档、路线图以及贡献指南。

## 🛠️ 技术栈 (Tech Stack)

* **前端**: Next.js 14, Tailwind CSS, Shadcn UI
* **合约**: Solidity, Foundry (测试框架)
* **通信**: Wagmi / Viem (Web3 交互)
* **存储**: IPFS (去中心化内容分发)

## 🤝 参与贡献 (Join the Build)
